# v1.11.11 · Initial public release 🎉

**DocCenter** — a local HTML workbench for browsing, editing & annotating all the HTML reports AI tools (Claude artifacts / ChatGPT canvas / Cursor / CodeBuddy) keep spitting out.

> Zero npm dependencies · Python 3 + aiohttp · Single file server · Runs at `http://localhost:9901`

---

## 为什么做这个工具？

过去一年我每天被 AI 生成的 HTML 文件淹没 —— 养虾系列、客户报告、思维导图、封面图……它们散落在十几个文件夹里，双击打开只能看，改一个错字得翻出原始 prompt 重跑一遍。

所以我写了 **DocCenter**：

- 🗂 **一站目录** — 扫描你指定的几个根目录，把所有 HTML 聚合成树
- ✏️ **所见即所得** — iframe 里的 HTML 直接可编辑，自动注入工具条（B/I/U/色板/对齐/链接）
- 💾 **自动快照 + 三选一保存** — 改完自动存到 `_auto-save/`，关闭时问你覆盖/另存/丢弃
- 🔒 **本地 first** — 127.0.0.1 only，不上云、不联网、文件始终在你硬盘上
- 🎨 **暗色模式 / 快捷键 / 时光机 / 面包屑** — 能想到的工作台该有的都有

---

## 🎯 核心场景

| 你在干 | DocCenter 帮你 |
|---|---|
| Claude artifacts 一天生成几十个 HTML | 自动聚合、浏览、打标签 |
| ChatGPT canvas 写完想改个措辞 | iframe 里直接改，自动快照兜底 |
| 团队内部 HTML 报告需要批注 | 高亮、加粗、色标，另存审阅版 |
| 多项目多根目录 | 多 scan_roots，设置面板白名单管理 |

## 🚀 Quick Start

```bash
git clone https://github.com/louisecxqiu-glitch/html-doc-center.git
cd html-doc-center
pip3 install aiohttp
python3 server.py
# → 打开 http://localhost:9901
```

macOS 开机自启：

```bash
cp launchd.plist.example ~/Library/LaunchAgents/com.louis.html-doc-center.plist
launchctl load ~/Library/LaunchAgents/com.louis.html-doc-center.plist
```

## 🛡 反 Bug 铁律（项目沉淀）

连续 10+ 版迭代后总结出的 5 条硬规矩，写在 [`ITERATION-SOP.md`](https://github.com/louisecxqiu-glitch/html-doc-center/blob/main/ITERATION-SOP.md)：

1. **真实浏览器演练** — curl 200 ≠ 用户视角能用
2. **守卫表达式必须显式验证** — IIFE 内 const 不会出现在 window 上
3. **CSS 改 .active / display 前 grep inline style 残留** — CSS 优先级干不过 inline
4. **DOM 切换后的依赖动作必须用 rAF** — 同步执行 = 在旧布局上算位置
5. **自驱模式 ≠ 跳过用户视角** — CHANGELOG 写得越漂亮越要真实验过

这些规矩是血泪经验，欢迎参考也欢迎挑战。

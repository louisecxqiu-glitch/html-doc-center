ARIS：对抗式 Multi-Agent 协作的技术拆解

🦞 养虾系列 · 第 18 篇 · 2026-05-18 · 路易乔布斯

一、问题定义

长 Agent 任务（8h+）会产生 "plausible unsupported success"——看似合理但缺乏证据支撑的成功。这是单 Agent 自审的根本缺陷：噪声 bandit 永远不会发现自己的盲点。

ARIS 的核心创新：将 质量保障 提升为一等工作流层。
通过跨家族对抗审查（Claude × GPT-5）打破 self-play 盲点。

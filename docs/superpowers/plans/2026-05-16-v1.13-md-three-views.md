## Goal

让 DocCenter 打开 `.md` 文件时，用户能够：

1. **拖拽中间分栏条**调整左侧编辑器和右侧预览的宽度比例（10%-90%）
2. **三档视图切换**：仅原文 / 分栏 / 仅预览（顶部工具栏按钮）
3. **状态持久化**到 localStorage，下次打开 `.md` 文件自动恢复（split ratio + view mode）

最终用户能感知的变化：

- 打开 `.md` 文件 → 顶部出现「📝 原文 | ⇔ 分栏 | 👁 预览」三档按钮
- 默认进分栏模式（保持当前 50:50 行为，不破坏现有体验）
- 鼠标移到中间分栏条上 → 光标变 `col-resize`，背景高亮
- 按住拖动 → 实时调整左右宽度（最小 10%、最大 90%）
- 点击"仅原文" → 左侧占满，右侧预览隐藏（但 DOM 保留以便 toggle）
- 点击"仅预览" → 右侧占满，左侧编辑器隐藏
- 关闭浏览器 → 下次打开任意 `.md` 文件，恢复上次的 viewMode 和 splitRatio
- 用户在编辑器输入 → 右侧预览实时（debounce 200ms）渲染

## Non-Goals

- ❌ **不做移动端响应式**（DocCenter 是 desktop 本地工具，触摸拖拽不在 scope）
- ❌ **不做垂直分栏**（仅左右分栏，不做上下分栏切换）
- ❌ **不做独立窗口/双屏分离**（预览独立窗口是 v2.0 级别能力）
- ❌ **不做键盘快捷键切换视图**（v1.14 再加，避免和 v1.11.7 时光机的快捷键冲突排查）
- ❌ **不改变现有非 .md 文件的渲染逻辑**（HTML 文件不受影响）
- ❌ **不做"按文件记忆"**——viewMode 和 splitRatio 全局共享（按文件记忆会让 localStorage 膨胀）
- ❌ **不引入 split.js / Allotment 等 npm 拖拽库**（保持零 npm 依赖约束）

### 3. JS 控制器（新文件 `web/md-view-controller.js`）

核心 API（约 80-100 LoC）：

```javascript
(function() {
  const LS_VIEW_MODE = 'doccenter.md.viewMode';     // 'source' | 'split' | 'preview'
  const LS_SPLIT_RATIO = 'doccenter.md.splitRatio'; // 0.1 ~ 0.9, default 0.5
  const MIN_RATIO = 0.1, MAX_RATIO = 0.9;

  function init(container) {
    const mode = localStorage.getItem(LS_VIEW_MODE) || 'split';
    const ratio = parseFloat(localStorage.getItem(LS_SPLIT_RATIO)) || 0.5;

    setViewMode(container, mode);
    setSplitRatio(container, ratio);

    // 按钮点击
    container.querySelectorAll('.md-view-btn').forEach(btn => {
      btn.addEventListener('click', () => setViewMode(container, btn.dataset.view));
    });

    // 拖拽
    const splitter = container.querySelector('.md-splitter');
    splitter.addEventListener('mousedown', e => startDrag(e, container));
  }

  function setViewMode(container, mode) {
    container.dataset.viewMode = mode;
    container.querySelectorAll('.md-view-btn').forEach(b => {
      b.classList.toggle('active', b.dataset.view === mode);
    });
    localStorage.setItem(LS_VIEW_MODE, mode);
  }

  function setSplitRatio(container, ratio) {
    ratio = Math.max(MIN_RATIO, Math.min(MAX_RATIO, ratio));
    container.style.setProperty('--md-split-left', (ratio * 100) + '%');
    container.style.setProperty('--md-split-right', ((1 - ratio) * 100) + '%');
    localStorage.setItem(LS_SPLIT_RATIO, ratio.toFixed(3));
  }

  function startDrag(e, container) {
    e.preventDefault();
    const splitter = container.querySelector('.md-splitter');
    splitter.classList.add('dragging');
    const body = container.querySelector('.md-body');
    const rect = body.getBoundingClientRect();

    function onMove(ev) {
      const ratio = (ev.clientX - rect.left) / rect.width;
      setSplitRatio(container, ratio);
    }
    function onUp() {
      splitter.classList.remove('dragging');
      document.removeEventListener('mousemove', onMove);
      document.removeEventListener('mouseup', onUp);
    }
    document.addEventListener('mousemove', onMove);
    document.addEventListener('mouseup', onUp);
  }

  window.mdViewCtl = { init, setViewMode, setSplitRatio };
})();
```

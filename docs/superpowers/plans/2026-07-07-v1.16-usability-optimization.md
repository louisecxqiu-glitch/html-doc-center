# HotPage 方便性优化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 让 HotPage 从"每次要 AI 帮忙启动"变成"双击即用、拖拽预览、一眼看到添加入口"。

**Architecture:** 纯前端改动为主（app.js + index.html + style.css），后端仅加一个 `--open-browser` 启动参数。拖拽预览走 FileReader + iframe.srcdoc，不经过后端，不改动安全模型。

**Tech Stack:** Vanilla JS（无框架）、aiohttp（后端）、HTML5 Drag & Drop API、FileReader API

if __name__ == "__main__":
    import argparse
    parser = argparse.ArgumentParser(description="HotPage — Local HTML workbench")
    parser.add_argument("--open-browser", action="store_true",
                        help="Automatically open browser after server starts")
    parser.add_argument("--port", type=int, default=9901,
                        help="Port to listen on (default: 9901)")
    args = parser.parse_args()

    # ... existing setup ...

    if args.open_browser:
        import threading, webbrowser
        def _open():
            time.sleep(1.5)
            webbrowser.open(f"http://localhost:{args.port}")
        threading.Thread(target=_open, daemon=True).start()

    web.run_app(app, host="127.0.0.1", port=args.port)

- [x] **Step 1: 在 app.js 加拖拽预览逻辑**

在 app.js 的初始化区域加：
```javascript
// ───────────── 拖拽 HTML 文件到主界面预览 ─────────────
  const mainArea = $("#main-area") || document.querySelector(".main-area") || document.body;
  let dragCounter = 0;

  mainArea.addEventListener("dragenter", (e) => {
    // 只处理文件拖入，忽略内部拖拽
    if (!e.dataTransfer?.types?.includes("Files")) return;
    e.preventDefault();
    dragCounter++;
    mainArea.classList.add("drag-over");
  });

  mainArea.addEventListener("dragleave", (e) => {
    e.preventDefault();
    dragCounter--;
    if (dragCounter <= 0) {
      dragCounter = 0;
      mainArea.classList.remove("drag-over");
    }
  });

  mainArea.addEventListener("dragover", (e) => {
    if (!e.dataTransfer?.types?.includes("Files")) return;
    e.preventDefault();
    e.dataTransfer.dropEffect = "copy";
  });

  mainArea.addEventListener("drop", async (e) => {
    if (!e.dataTransfer?.files?.length) return;
    e.preventDefault();
    dragCounter = 0;
    mainArea.classList.remove("drag-over");

    const files = Array.from(e.dataTransfer.files);
    // 优先找 HTML/MD 文件
    const target = files.find(f => /\.html?$/i.test(f.name))
                || files.find(f => /\.md$/i.test(f.name));
    if (!target) {
      toast("请拖入 HTML 或 Markdown 文件", "warning");
      return;
    }

    // 读取文件内容
    const text = await target.text();
    const isMD = /\.md$/i.test(target.name);

    // 渲染到 iframe
    const iframe = $("#file-frame");
    if (isMD) {
      // MD 文件：走现有的 MD 渲染逻辑（如果有）
      // 简单方案：用 srcdoc 包一层
      iframe.srcdoc = text;
    } else {
      // HTML 文件：直接 srcdoc
      iframe.srcdoc = text;
    }

    // 更新面包屑显示文件名
    const bc = $("#breadcrumb");
    bc.innerHTML = `<b>${escapeHtml(target.name)}</b> <span class="bc-meta" style="color:#999">📋 拖入预览（未保存）</span>`;

    toast(`已加载：${target.name}`, "success");
  });
})();
```

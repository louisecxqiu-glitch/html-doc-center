# MVP Phase 1 — 画像录入 + AI 生成暑假计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 端到端可运行的最小 MVP：用户在网页上录入孩子画像+家长偏好 → 点击生成 → AI 返回个性化暑假计划（含时间表+书单+教辅推荐）。

**Architecture:** FastAPI 后端（端口 9902）提供 `/api/generate-plan` 接口，内部调用 DeepSeek-V3 生成计划。前端为单个 HTML 页面，含表单录入 + 结果展示。知识库用 JSON 文件。

**Tech Stack:** Python 3 + FastAPI + openai（DeepSeek 兼容）+ 纯 HTML/CSS/JS

## Global Constraints

- API key 从环境变量读取，不硬编码
- HTML 正文禁止 Markdown 语法
- 禁止 opacity 弱化
- 文档语言：简体中文
- 代码注释用英文，用户可见文案用中文
- DeepSeek-V3（deepseek-chat），temperature=0.3

- [ ] **Step 2: 创建 prompts.py**

系统提示词 + 计划生成 prompt 模板。核心逻辑：接收画像 JSON + 知识库数据 → 构造 prompt → 要求 AI 返回结构化 JSON。

- [ ] **Step 3: 创建 main.py**

FastAPI 应用，提供：
- `GET /` → 返回 `static/index.html`
- `GET /api/books` → 返回书单知识库
- `GET /api/workbooks` → 返回教辅知识库
- `POST /api/generate-plan` → 接收画像 JSON，调用 DeepSeek 生成计划，返回结构化结果

- [ ] **Step 4: 安装依赖并启动验证**

运行 `pip3 install fastapi uvicorn`，然后 `uvicorn main:app --port 9902`，curl 验证 `/api/books` 返回 200。

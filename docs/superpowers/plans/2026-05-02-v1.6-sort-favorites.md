# v1.6.0 — 目录排序 + 文件收藏 + Disabled 目录沉底

## Goal

3 个相关的目录树体验优化：

1. **目录排序**：每层独立排序，支持 "修改时间（默认倒序）/ 名称"，偏好持久化
2. **文件/目录收藏**：文件名旁 ★ 图标点击收藏，侧边栏顶部展示"⭐ 收藏"分组
3. **Disabled scan_roots 沉底**：关闭的扫描目录自动排在后面，视觉不抢眼

## Architecture

- 后端 `server.py`：`_walk_dir` 加 sort 参数；新增 `GET|POST /api/favorites` 读写 `~/.codebuddy/html-doc-center/favorites.json`
- 前端 `app.js`：侧边栏顶部新增"⭐ 收藏"分组、每个 file-label 加 ☆ 按钮；排序切换按钮（搜索框旁下拉）；scan_roots 列表按 enabled 分组渲染
- 收藏数据单独 `favorites.json`，不污染 config.json
- 跨目录收藏允许，但必须落在 scan_roots 白名单内（通过 `_resolve_safe` 校验）
- 失效收藏自动清理 + toast 提示

## Tech Stack

Python 3 + aiohttp · 原生 JS · 无新增依赖
